## DESIGN NOTES FOR AVADA PAGE BUILDER:

### Color Palette Recommendations:
- **Primary Color:** Trust-building blue or calming green (for main CTAs, headers)
- **Accent Color:** Energetic orange or warm red (for highlights, icons)
- **Background:** Clean white with subtle texture
- **Text:** Dark gray (#333) for readability

### Typography:
- **Headlines:** Bold, friendly sans-serif (Montserrat, Poppins, Open Sans)
- **Body Text:** Readable serif or clean sans-serif (Lora, Roboto, Source Sans)
- **Size hierarchy:** Clear distinction between H1 (40-48px) ‚Üí H2 (32-36px) ‚Üí H3 (24-28px) ‚Üí Body (16-18px)

### Images Needed:
1. Hero image: Owner with attentive dog (eye contact moment)
2. Transformation steps: 3 simple icons or illustrations
3. Pam's professional headshot (warm, approachable)
4. Testimonial photos (if available, or use stock images of happy dog owners)
5. Comparison table icons
6. Trust badges/certifications
7. Section background patterns (subtle, not distracting)

### Mobile Optimization:
- Stack all multi-column sections vertically on mobile
- Make CTA buttons full-width and thumb-friendly
- Reduce hero text size for mobile screens
- Ensure comparison table is horizontally scrollable or reformatted for mobile
- Touch-friendly FAQ accordions

### Conversion Elements:
- Sticky header with CTA button on scroll
- Exit-intent popup (optional): "Wait! Get a free recall training tip before you go"
- Social proof notifications (optional): "Sarah from [City] just enrolled!"
- Progress bar showing how far down the page visitor has scrolled
- Multiple CTAs throughout page (minimum 4-5 opportunities to purchase)

### Page Load Optimization:
- Lazy load images below the fold
- Compress all images (WebP format preferred)
- Minimize animations (only use subtle fades/slides)
- Prioritize above-the-fold content loading
