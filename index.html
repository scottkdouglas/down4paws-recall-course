RULE #1

Never Call for Unpleasant Things

+

Stop calling your dog to clip their nails, give them a bath, or leave the dog park. You're teaching them that "come" = something bad. Use their name or go get them instead.

RULE #2

Only Call When You're Sure They'll Come

+

If your dog is chasing a squirrel, don't call them. You're practicing failure. Set them up to succeed - call when distractions are low, then gradually increase difficulty.

RULE #3

Make It Happen Every Time

+

If you call and they don't come, calmly go get them. Never let "come" be optional. This single rule changes everything.

RULE #4

Never Repeat the Command

+

Saying "come, come, COME, COME!" teaches your dog to ignore you the first 3 times. One call. Then go help them succeed.

RULE #5

Pay Your Dog What They're Worth

+

Real talk - you need to figure out what your dog actually wants. For some dogs it's hot dogs. For others it's a ball. I'll show you how to run "food auditions" to discover your dog's currency.
